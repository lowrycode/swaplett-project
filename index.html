INSTRUCTIONS

Aim of the Game

Swap the letters into the correct positions in the words (by dragging one letter onto another) within the specified number of swaps.

Words are arranged in the grid both horizontally and vertically.

  • GREEN - the letter is in the correct position and will no longer be movable
  • YELLOW - the letter is found somewhere else in the same row or column

Before The Game

Choose grid size

This determines the shape of the grid and how many letters are in each word.

Choose difficulty level

Choose from one of the options below:

  • EASY - can be solved within 6 swaps
  • MEDIUM - can be solved within 8 swaps
  • HARD - can be solved within 10 swaps

After The Game

Scroll down to view the definitions of the words. If available, you can listen to an audio clip of the word by pressing the icon.

Click the PLAY AGAIN button.

This game is inspired by a popular game called Waffle and was built as a portfolio project for the Diploma in Full Stack Development course with Code Institute.
